# MoodSync

MoodSync is a mood-adaptive mobile application developed to support mental well-being through emotion tracking and personalised recommendations. The system detects user moods using AI techniques and adapts content and UI accordingly.

This project was developed as part of the **Software Development Group Project (SDGP)** at the **University of Westminster**.

---

## Features

- Text and voice-based mood detection  
- Mood history tracking  
- Mood-based music and food recommendations  
- Adaptive UI and theme customisation  
- Inspirational quotes and activities  
- Anonymous community story sharing  

---

## Technology Stack

**Frontend**
- Flutter (Android & iOS)

**Backend**
- FastAPI (Python)
- PostgreSQL
- REST APIs
- JWT Authentication

**Machine Learning**
- NLP emotion detection (GoEmotions)
- Speech Emotion Recognition (RAVDESS)

**APIs & Services**
- YouTube Music API  
- Spoonacular API  
- Firebase  
- Unsplash API
